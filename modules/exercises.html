Question 01: Find the maximum yield length for the following CNF CFG:

S ‚Üí AB
A ‚Üí a
B ‚Üí b
üìò SOLUTION (Click to Expand)

Derivation Tree for the string ab

  • Longest path length = 2 = k

  • Maximum yield length = 2^(k ‚àí 1) = 2

Concept Insight

The yield length corresponds to the number of terminals produced at the leaf level. In CNF, each derivation step can at most double the number of leaves, hence the formula \(2^{k-1}\) for the maximum yield length.
