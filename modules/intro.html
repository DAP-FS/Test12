1. Best Practices

Question 1: Construct a grammar that accepts the set of all strings of length 2 (\(\Sigma = \{a, b\}\)).

Solution (hidden):

Show Solution
\[\begin{split}S \to X X \\ X \to a \mid b\end{split}\]

Step-by-step (easy):

  1. We want strings of exactly length 2.

  2. Introduce a symbol \(X\) that generates a single character (\(a\) or \(b\)).

  3. The start symbol \(S\) should produce two \(X\) in sequence, so \(S \to X X\).

  4. This allows all four strings: \(aa, ab, ba, bb\).

    </details>

Question 2: Grammar for \(L = \{ a^n b^m \mid n \ge 1, m \ge 1 \}\).

Solution (hidden):

Show Solution
\[\begin{split}S \to A B \\ A \to a A \mid a \\ B \to b B \mid b\end{split}\]

Step-by-step (easy):

  1. The language is some positive number of \(a\) followed by some positive number of \(b\).

  2. Use \(A\) to generate one or more \(a\)’s (\(n \ge 1\)): \(A \to a A \mid a\).

  3. Use \(B\) to generate one or more \(b\)’s (\(m \ge 1\)): \(B \to b B \mid b\).

  4. The start symbol \(S\) concatenates them: \(S \to A B\).

  5. Example: to produce \(aaab\), expand \(S \to A B\), \(A \to a A \to a a A \to a a a\), and \(B \to b\).

    </details>

Question 3: Grammar for \(L = \{ a^n b^n \mid n \ge 0 \}\).

Solution (hidden):

Show Solution
\[S \to a S b \mid \epsilon\]

Step-by-step (easy):

  1. We must produce equal numbers of \(a\) and \(b\), with all \(a\) before \(b\).

  2. The rule \(S \to a S b\) adds one \(a\) at the left and one \(b\) at the right together — that keeps the counts equal.

  3. The base \(S \to \epsilon\) handles the case \(n = 0\) (empty string).

  4. Example: For \(n=2\), expand \(S \to a S b \to a a S b b \to a a \epsilon b b = a a b b\).

    </details>

Question 4: Set of strings of length at least 2 (\(\Sigma = \{a, b\}\)).

Solution (hidden):

Show Solution
\[\begin{split}S \to X X Y \\ X \to a \mid b \\ Y \to a Y \mid b Y \mid \epsilon\end{split}\]

Step-by-step (easy):

  1. We must generate any string whose length is ≥ 2.

  2. Ensure at least two characters by requiring two occurrences of \(X\) at the start: \(S \to X X Y\).

  3. \(X\) generates a single symbol (\(a\) or \(b\)). So \(X X\) guarantees two symbols minimum.

  4. \(Y\) generates the rest of the string, which can be any length (including zero) from the set \(\Sigma^*\). The rules for \(Y\) are equivalent to \((a \mid b)^*\).

    </details>

Question 5: Set of strings of length at most 2 (\(\Sigma = \{a, b\}\)).

**Solution (hidden): >

Show Solution
\[\begin{split}S \to \epsilon \mid X \mid X X \\ X \to a \mid b\end{split}\]

Step-by-step (easy):

  1. Strings allowed: length 0, 1, or 2.

  2. Use the alternatives: empty string (\(\epsilon\)), one symbol (\(X\)), or two symbols (\(X X\)).

  3. Define \(X\) as a single letter \(a\) or \(b\).

  4. This covers \(\{\epsilon, a, b, aa, ab, ba, bb\}\).

    </details>

Question 6: Set of strings starting with \(a\) and ending with \(b\) (\(\Sigma = \{a, b\}\)).

Solution (hidden):

Show Solution
\[\begin{split}S \to a X b \mid a b \\ X \to a X \mid b X \mid \epsilon\end{split}\]

Step-by-step (easy):

  1. Every string must begin with \(a\) and end with \(b\).

  2. The base case is the minimal string \(ab\), handled by \(S \to a b\).

  3. For longer strings, use \(S \to a X b\), where \(X\) generates any string of length $ge 1$.

  4. The non-terminal \(X\) generates any string from \(\Sigma^*\) using the rules \(X \to a X \mid b X \mid \epsilon\). The overall grammar must include the minimal string \(ab\).

    </details>

Question 7: Set of strings starting and ending with different symbols (\(\Sigma = \{a, b\}\)).

Solution (hidden):

Show Solution
\[\begin{split}S \to A \mid B \\ A \to a X b \\ B \to b X a \\ X \to a X \mid b X \mid a \mid b \mid \epsilon\end{split}\]

Step-by-step (easy):

  1. The language is the union of two cases: (Starts with \(a\) and ends with \(b\)) OR (Starts with \(b\) and ends with \(a\)).

  2. Use \(S \to A \mid B\) to separate the cases.

  3. \(A \to a X b\) covers the first case. \(B \to b X a\) covers the second case.

  4. The non-terminal \(X\) generates any string from \(\Sigma^*\), ensuring all strings of length $ge 2$ (e.g., \(ab\), \(ba\)) are covered. The rule \(X \to \epsilon\) handles the length 2 strings, while \(X \to a \mid b\) handles length 3 strings like \(aab\) and \(baa\).

    </details>

Question 8: Set of all strings starting and ending with the same symbol (\(\Sigma = \{a, b\}\)).

Solution (hidden):

Show Solution
\[\begin{split}S \to a X a \mid b X b \mid a \mid b \mid \epsilon \\ X \to a X \mid b X \mid \epsilon\end{split}\]

Step-by-step (easy):

  1. This includes four sub-cases:

    1. Strings of length $ge 2$ starting and ending with \(a\): \(a X a\).

    2. Strings of length $ge 2$ starting and ending with \(b\): \(b X b\).

    3. Strings of length 1 (\(a\) or \(b\)).

    4. String of length 0 (\(\epsilon\)).

  2. The production \(S \to a \mid b \mid \epsilon\) handles cases (c) and (d).

  3. The productions \(S \to a X a \mid b X b\) handle cases (a) and (b).

  4. \(X \to a X \mid b X \mid \epsilon\) generates any possible middle section \(\Sigma^*\).

    </details>

Question 9: Grammar for \(L = \{ a^n b^n c^m \mid n \ge 0, m \ge 0 \}\).

Solution (hidden):

Show Solution
\[\begin{split}S \to A B \\ A \to a A b \mid \epsilon \\ B \to c B \mid \epsilon\end{split}\]

Step-by-step (easy):

  1. The language is a sequence of equal \(a\)’s and \(b\)’s, followed by any number of \(c\)’s. This suggests concatenation of two independent parts.

  2. Use non-terminal \(A\) for the equal \(a^n b^n\) part (CFG is required here): \(A \to a A b \mid \epsilon\).

  3. Use non-terminal \(B\) for the arbitrary \(c^m\) part (Regular Grammar is sufficient): \(B \to c B \mid \epsilon\).

  4. Concatenate them: \(S \to A B\).

    </details>

Question 10: Set of all palindromes (\(w = w^R, \Sigma = \{a, b\}\)).

Solution (hidden):

Show Solution
\[S \to a S a \mid b S b \mid a \mid b \mid \epsilon\]

Step-by-step (easy):

  1. Palindromes must read the same forward and backward.

  2. The recursive rules \(S \to a S a \mid b S b\) ensure that whatever is added to the left is also added to the right, preserving the palindrome property.

  3. The base cases cover:

    1. Odd-length palindromes (middle symbol): \(S \to a \mid b\).

    2. Even-length palindromes (empty middle): \(S \to \epsilon\).

  4. Example: \(aba\) from \(S \to a S a \to a b a\). Example: \(abba\) from \(S \to a S a \to a b S b a \to a b \epsilon b a = a b b a\).

    </details>

Question 11: Even length strings (\(|w| = 2k, k \ge 0, \Sigma = \{a, b\}\)).

Solution (hidden):

Show Solution
\[\begin{split}S \to X X S \mid \epsilon \\ X \to a \mid b\end{split}\]

Step-by-step (easy):

  1. We want strings where the length is a multiple of 2 (0, 2, 4, …).

  2. The recursive rule must add symbols in pairs.

  3. \(S \to X X S\) adds two symbols (\(X X\)) and continues the process (\(S\)).

  4. The base case \(S \to \epsilon\) stops the recursion, resulting in an even length.

  5. \(X \to a \mid b\) generates a single character.

    </details>

Question 12: Grammar for \(L = \{ w \mid |w| \equiv 0 \pmod 3, \Sigma = \{a, b\} \}\). (Length is a multiple of 3)

Solution (hidden):

Show Solution
\[\begin{split}S \to X X X S \mid \epsilon \\ X \to a \mid b\end{split}\]

Step-by-step (easy):

  1. We want strings where the length is a multiple of 3 (0, 3, 6, 9, …).

  2. The recursive rule must add symbols in triplets.

  3. \(S \to X X X S\) adds three symbols (\(X X X\)) and continues the process (\(S\)).

  4. The base case \(S \to \epsilon\) stops the recursion, resulting in a length divisible by 3.

  5. \(X \to a \mid b\) generates a single character.

    </details>

Question 13: Grammar for \(L = \{ a^n b^n c^n \mid n \ge 0 \}\).

**Solution (hidden):</b>

Show Solution

Not possible with a Context-Free Grammar (CFG).

Step-by-step (easy):

  1. The language requires three separate components to have equal counts: \(Count(a) = Count(b) = Count(c)\).

  2. A CFG pushdown automaton (PDA) can only simultaneously enforce the equality of two counts (e.g., \(a^n b^n\)) using its single stack.

  3. To enforce the third equality (\(Count(c)=n\)), the PDA would need a second stack.

  4. Since \(L = \{ a^n b^n c^n \mid n \ge 0 \}\) is a Context-Sensitive Language (CSL), it cannot be generated by a CFG.

    </details>

Question 14: Grammar for \(L = \{ a^n b^m \mid n > m, n \ge 0, m \ge 0 \}\). (More \(a\)’s than \(b\)’s)

**Solution (hidden):</b>

Show Solution
\[\begin{split}S \to A B \\ A \to a A b \mid a \\ B \to a B \mid \epsilon\end{split}\]

Step-by-step (easy):

  1. The language requires a minimal string of one extra \(a\) (\(a^{m+1} b^m\)), followed by any number of additional \(a\)’s.

  2. The CFG needs to first match \(a^m b^m\) and then ensure the single extra \(a\).

  3. Use non-terminal \(A\) to enforce the at least one extra \(a\): \(A \to a A b\) matches the $m$ pairs, and the minimal production \(A \to a\) guarantees the one extra $a$.

  4. Use non-terminal \(B\) to generate any arbitrary number of $a$’s at the end (the $a$’s not needed for matching $b$’s): \(B \to a B \mid \epsilon\).

  5. Concatenate them: \(S \to A B\).

    </details>

Question 15: Grammar for \(L = \{ a^n b^m \mid n \ne m, n \ge 0, m \ge 0 \}\). (Unequal numbers of \(a\)’s and \(b\)’s)

**Solution (hidden):</b>

Show Solution
\[\begin{split}S \to A \mid B \\ A \to a A b \mid a C \\ B \to a B b \mid b C \\ C \to a C \mid b C \mid \epsilon\end{split}\]

Step-by-step (easy):

  1. The language is the union of two independent cases: Case A: More :math:`a`’s than :math:`b`’s (covered by $A$) OR Case B: More :math:`b`’s than :math:`a`’s (covered by $B$).

  2. Case A (n > m): Match the pairs with \(A \to a A b\), and then use \(A \to a C\) to enforce one extra \(a\) followed by any remaining \(a\)’s or \(b\)’s generated by \(C\).

  3. Case B (m > n): Match the pairs with \(B \to a B b\), and then use \(B \to b C\) to enforce one extra \(b\) followed by any remaining symbols.

  4. \(C \to a C \mid b C \mid \epsilon\) generates any string \(\Sigma^*\).

    </details>

Question 16: Grammar for \(L = \{ a^n b^n c^m d^m \mid n \ge 0, m \ge 0 \}\).

**Solution (hidden):</b>

Show Solution
\[\begin{split}S \to A B \\ A \to a A b \mid \epsilon \\ B \to c B d \mid \epsilon\end{split}\]

Step-by-step (easy):

  1. The language consists of two independent parts concatenated: \((a^n b^n)\) followed by \((c^m d^m)\).

  2. Use \(A\) for the first part: \(A \to a A b \mid \epsilon\). This ensures \(Count(a) = Count(b)\).

  3. Use \(B\) for the second part: \(B \to c B d \mid \epsilon\). This ensures \(Count(c) = Count(d)\).

  4. Since the two parts are independent and adjacent, a CFG can handle it: \(S \to A B\). This language is the intersection of two CFGs, but since they are distinct parts of the string, the grammar is simple.

    </details>

Question 17: Grammar for \(L = \{ w w^R \mid w \in \{a, b\}^* \}\). (Even-length palindromes)

**Solution (hidden):</b>

Show Solution
\[S \to a S a \mid b S b \mid \epsilon\]

Step-by-step (easy):

  1. This is the subset of palindromes that must have an even length (since \(w w^R\) is always $2|w|$).

  2. The recursive rules \(S \to a S a \mid b S b\) enforce the palindrome structure by adding identical symbols to the ends.

  3. The only base case is the empty string \(S \to \epsilon\), which correctly generates the smallest even-length palindrome (length 0).

  4. The rules from Question 10 (Palindromes) without the $S to a mid b$ productions are exactly what is needed here.

    </details>

Question 18: Grammar for \(L = \{ a^i b^j c^k \mid i = j \text{ or } j = k \}\).

**Solution (hidden):</b>

Show Solution
\[\begin{split}S \to A \mid B \\ A \to C D \\ C \to a C b \mid \epsilon \\ D \to c D \mid \epsilon \\ B \to E F \\ E \to a E \mid \epsilon \\ F \to b F c \mid \epsilon\end{split}\]

Step-by-step (easy):

  1. The language is the union of two CFGs: Case A: \(i = j\) (arbitrary \(k\)) OR Case B: \(j = k\) (arbitrary \(i\)).

  2. \(S \to A \mid B\) separates the two cases.

  3. Case A (:math:`i=j`): \(A \to C D\). $C$ generates \(a^i b^i\) ($C to a C b mid epsilon$) and $D$ generates arbitrary $c$’s ($D to c D mid epsilon$).

  4. Case B (:math:`j=k`): \(B \to E F\). $E$ generates arbitrary $a$’s ($E to a E mid epsilon$) and $F$ generates \(b^j c^j\) ($F to b F c mid epsilon$).

    </details>

Question 19: Grammar for \(L = \{ a^n b^n c^m d^m \mid n \ge 1, m \ge 1 \}\).

**Solution (hidden):</b >

Show Solution
\[\begin{split}S \to A B \\ A \to a A b \mid a b \\ B \to c B d \mid c d\end{split}\]

Step-by-step (easy):

  1. This is almost identical to Question 16, but with the constraint that $n ge 1$ and $m ge 1$ (no empty strings for $a^n b^n$ or $c^m d^m$).

  2. Use \(A\) for the \(a^n b^n\) part, guaranteeing $n ge 1$: \(A \to a A b \mid a b\).

  3. Use \(B\) for the \(c^m d^m\) part, guaranteeing $m ge 1$: \(B \to c B d \mid c d\).

  4. Concatenate them: \(S \to A B\).

    </details>

Question 20: Grammar for \(L = \{ a^n b^n c^n \mid n \ge 1 \}\).

**Solution (hidden):</b>

Show Solution

Not possible with a Context-Free Grammar (CFG).

Step-by-step (easy):

  1. This is the same language as Question 13, but excludes the empty string (\(n=0\)).

  2. The core constraint \(Count(a) = Count(b) = Count(c)\) requires the PDA to track three counts simultaneously, which is impossible with a single stack.

  3. This language is a classic example used in the Pumping Lemma for CFGs to prove it is not context-free.

    </details>

Question 21: Grammar for \(L = \{ w \mid w \text{ has equal numbers of } a\text{'s and } b\text{'s} \}\).

**Solution (hidden):</b>

Show Solution
\[S \to S S \mid a S b \mid b S a \mid \epsilon\]

Step-by-step (easy):

  1. This is a classic non-regular CFG where the order of \(a\)’s and \(b\)’s does not matter, only that they cancel out to zero difference.

  2. \(S \to S S\) allows concatenation (e.g., \(aabb\) followed by \(bbaa\) is valid).

  3. \(S \to a S b\) and \(S \to b S a\) ensure that one \(a\) and one \(b\) are generated together, preserving the equality count.

  4. \(S \to \epsilon\) is the base case (empty string is valid).

  5. This grammar generates strings like \(ab, ba, aabb, abab, abba, baab\), etc., all with $N_a = N_b$.

    </details>

Question 22: Grammar for \(L = \{ w \mid N_a(w) > N_b(w) \}\). (More \(a\)’s than \(b\)’s, any order)

**Solution (hidden):</b>

Show Solution
\[\begin{split}S \to A \\ A \to a \mid a A \mid b A a A \mid a A b A\end{split}\]

Step-by-step (easy):

  1. This is a well-known example that is a CFG, but complex to construct. The goal is to ensure the final string has a net surplus of one or more \(a\)’s.

  2. We use recursion to ensure that all balanced pairs of \(a\)’s and \(b\)’s are generated, and a single \(a\) is left over.

  3. The rule \(A \to a\) ensures the minimum of one extra \(a\).

  4. The rules \(A \to a A\), \(A \to a A b A\), and \(A \to b A a A\) ensure that we can generate any arbitrary string while maintaining the overall property of $N_a > N_b$. (A simpler form using an intermediate non-terminal is also possible but often more complex to prove correct.)

    </details>

Question 23: CYK Algorithm Question: Check whether the string \(w=aab\) is a valid member of the grammar \(G\) where the productions are: \(S \to AB \mid BC\), \(A \to BA \mid a\), \(B \to CC \mid b\), \(C \to AB \mid a\).

CYK Table:

The CYK table for \(w = a_1 a_2 b_3\) (length 3):

Length $l$ | 1 | 2 | 3 |
:—: | :—: | :—: | :—: |
$V_{i, 1}$ | \(V_{1,1}\) ($a$) | \(V_{2,1}\) ($a$) | \(V_{3,1}\) ($b$) |
Sets | \(\{A, C\}\) | \(\{A, C\}\) | \(\{B\}\) |
$V_{i, 2}$ | \(V_{1,2}\) ($aa$) | \(V_{2,2}\) ($ab$) | |
Sets | \(\emptyset\) | \(\{S, A\}\) | |
$V_{1, 3}$ | \(V_{1,3}\) ($aab$) | | |
Sets | \(\emptyset\) | | |

Detailed Calculation:

  1. Length 1 ($V_{i, 1}$): * $V_{1, 1}$ (for $a$): ${A, C}$ (from $A to a, C to a$) * $V_{2, 1}$ (for $a$): ${A, C}$ * $V_{3, 1}$ (for $b$): ${B}$ (from $B to b$)

  2. Length 2 ($V_{i, 2}$): * $V_{1, 2}$ (for $a_1 a_2 = aa$): Check $V_{1, 1} V_{2, 1} = {A, C} {A, C}$. Productions $AA, AC, CA, CC$. Only $B to CC$ exists. $implies V_{1, 2} = {B}$. (Correction: checking $V_{1, 1} V_{2, 1}$ gives $AA, AC, CA, CC$. The given grammar only has $S to AB, BC$ etc.)

    • Let’s re-check $V_{1, 2}$: $V_{1, 1} V_{2, 1} = {A, C} {A, C}$.

    • $A A$: Not possible.

    • $A C$: Not possible.

    • $C A$: Not possible.

    • $C C$: Possible from $B to C C$. $implies V_{1, 2} = {B}$. (My earlier table entry was wrong).

    • $V_{2, 2}$ (for $a_2 b_3 = ab$): Check $V_{2, 1} V_{3, 1} = {A, C} {B}$. Productions $A B, C B$.

      • $A B$: Possible from $S to A B, C to A B$. $implies {S, C}$.

      • $C B$: Not possible.

      • $implies V_{2, 2} = {S, C}$.

  3. Length 3 ($V_{1, 3}$): Check all splits (k=1, 2): * Split 1: $V_{1, 1} V_{2, 2} = {A, C} {S, C}$. Productions $A S, A C, C S, C C$.

    • $C C$: Possible from $B to C C$. $implies {B}$

    • Split 2: $V_{1, 2} V_{3, 1} = {B} {B}$. Productions $B B$.

      • $B B$: Not possible.

    • $implies V_{1, 3} = {B}$.

Conclusion:

Since the start symbol \(S\) is not in the set \(V_{1, 3}\) (the set for the full string $aab$), the string \(aab\) is NOT a valid member of the language generated by grammar \(G\).
