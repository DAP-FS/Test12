BASIC

  • Input: \(\Sigma = \{a, b\}\)

  • Stack: \(\Gamma = \{Z_0, A\}\)

Œ¥(q‚ÇÄ, a, Z‚ÇÄ) = (q‚ÇÄ, AZ‚ÇÄ) // First 'a'

Question 15: Prove that \(L = \{a^n b^n c^n \mid n \geq 1\}\) is NOT context-free.

ADVANCED

Solution:

üîç Show Complete Solution
NOT POSSIBLE with a PDA
üéØ Proof Using Pumping Lemma:
  1. The Problem: Need to enforce three equal counts simultaneously
  2. PDA Limitation: A PDA has only ONE stack
  3. What PDAs Can Do: Match TWO counts (like anbn)
  4. Why Three Fails:
    • To match a's and b's, we push 'a's and pop for 'b's
    • But then we can't also verify c's match without a second stack
    • Would need 2D memory (two stacks or equivalent)
  5. Formal Proof: Use Pumping Lemma for CFLs:
    • Let p be pumping length, consider w = apbpcp
    • Any decomposition w = uvxyz with |vxy| ‚â§ p
    • vxy spans at most 2 symbol types
    • Pumping uv¬≤xy¬≤z breaks three-way equality
  6. Language Class: This is Context-Sensitive (needs LBA)
üí≠ Key Insight: A PDA can "count" one relationship using its stack. To count two independent relationships simultaneously, you need more power than a PDA provides.

Question 16: Prove that \(L = \{ww \mid w \in \{a,b\}^*\}\) is NOT context-free.

ADVANCED

Solution:

üîç Show Complete Solution
NOT POSSIBLE with a PDA
üéØ Why Copying Fails:
  1. The Challenge: Need to verify second half is EXACT copy of first
  2. Difference from wwR:
    • wwR (palindrome): Can use stack to verify reverse
    • ww (copy): Would need to verify exact forward copy
  3. Why PDA Fails:
    • LIFO (Last-In-First-Out) structure of stack
    • Can only compare in reverse order
    • Cannot "remember" first half in forward order AND compare
  4. Pumping Lemma Proof:
    • Consider w = apbpapbp
    • Any pumping will break the ww structure
    • Cannot maintain exact duplication
Contrast: wwR is CFL but ww is NOT CFL. The difference is in the ORDER of comparison - reverse vs forward!

Question 17: Prove that \(L = \{a^{n^2} \mid n \geq 1\}\) is NOT context-free.

ADVANCED

Solution:

üîç Show Complete Solution
NOT POSSIBLE with a PDA
üéØ Non-Linear Count Problem:
  1. Language: Strings with 1, 4, 9, 16, 25,... a's (perfect squares)
  2. Why PDA Fails:
    • PDAs can handle LINEAR relationships (anbn)
    • Cannot handle QUADRATIC or higher polynomial growth
    • Stack can only count linearly
  3. Pumping Lemma Application:
    • Let p be pumping length
    • Consider w = ap¬≤
    • Pumping changes count by linear amount
    • p¬≤ + k is not a perfect square for small k
üí≠ General Rule: Languages with non-linear counting (exponential, factorial, prime, perfect squares) are NOT context-free.

Summary: What Makes Languages Non-Context-FreeÔÉÅ

üéØ Patterns That CANNOT Be Handled by PDAs:

1. Three or More Equal Counts:
‚Ä¢ anbncn - needs to track three relationships
‚Ä¢ Requires more than one stack

2. Forward Copying:
‚Ä¢ ww - needs to remember and compare in same direction
‚Ä¢ wn - multiple forward copies
‚Ä¢ Stack is LIFO, can only compare in reverse

3. Arithmetic Properties:
‚Ä¢ Primes: an where n is prime
‚Ä¢ Perfect squares: an¬≤
‚Ä¢ Factorials: an!
‚Ä¢ Exponentials: an^m
‚Ä¢ Requires divisibility testing or complex arithmetic

‚Ä¢ aibjckdl where i=k AND j=l
‚Ä¢ Two independent equalities that "cross" each other

5. Mixed Constraints:
‚Ä¢ wwwR - combines copying with reverse
‚Ä¢ Multiple simultaneous conditions
‚ö° Why These Fail:

PDA Limitations:
‚Ä¢ Only ONE stack (LIFO structure)
‚Ä¢ Can track at most ONE relationship
‚Ä¢ Can compare only in REVERSE order
‚Ä¢ Cannot perform arithmetic beyond counting

What's Needed:
‚Ä¢ Two stacks ‚Üí Linear Bounded Automaton (LBA)
‚Ä¢ Full tape ‚Üí Turing Machine
‚Ä¢ Multiple simultaneous counters ‚Üí Beyond CFLs
üéì Proof Techniques:
  1. Pumping Lemma for CFLs: Show no matter how you pump, language membership breaks
  2. Closure Properties: Show if L were CFL, then intersecting with regular language gives known non-CFL
  3. Reduction: Show if L were CFL, then another known non-CFL would also be CFL (contradiction)
  4. Direct Argument: Show inherent limitation of PDA (like LIFO for ww)

üéØ Essential Patterns:
